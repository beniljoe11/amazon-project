# Amazon Sales Analytics Dashboard

## Overview
This project is an interactive Power BI dashboard built to analyze Amazon sales data and generate business insights.

## Features
- Sales trend analysis
- Profit analysis by state
- Category-wise sales breakdown
- KPI tracking dashboard

## Tools Used
- Power BI
- Excel / CSV
- Data Visualization

## Key Metrics
- Total Orders: 1611
- Total Sales: 725.46K
- Total Profit: 108.42K

## Project Purpose
This project demonstrates data analysis, dashboard development, and business intelligence skills for Data Analyst and Power BI roles.
